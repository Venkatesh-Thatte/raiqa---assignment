# Home Chef Meals

A simple meal browsing and selection app built with Next.js.

## Features
- Browse meals from home chefs with name, price, and availability
- Toggle between available-only and all meals
- Add available meals to a selection (no duplicates)
- View selected meals with a live total price
- Sort meals by price (Low → High / High → Low)
- Highlights cheapest and priciest selected meals
- Reset selection with one click
- Persists selection across page refreshes via localStorage

## Tech Stack
- Next.js 14 (App Router)
- React with useState & useEffect
- Plain CSS (no UI library)

## Getting Started
npm install
npm run dev
